# L-Hub for Codex Downloads

Public installer and release artifacts for `L-Hub for Codex`.

This repository does not contain the private source code. It only hosts versioned install assets for end users.

## Install

```bash
curl -fsSL https://github.com/readysteadyscience/l-hub-for-codex-downloads/raw/refs/heads/main/install.sh | bash
```

After installation:

```bash
lhub console
lhub init
lhub doctor
```
